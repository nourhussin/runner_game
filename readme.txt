==============================
      Runner Game
==============================

This is a simple runner game made with Python using the Pygame Zero (pgzrun) library.

------------------------------
How to Play
------------------------------
1. Open your terminal or command prompt.
2. Navigate to the folder where your game.py file is located.
3. Run the game using Pygame Zero by typing:
   pgzrun game.py
4. Controls:
   - Left Arrow: Move player left
   - Right Arrow: Move player right
   - Avoid obstacles and monsters
   - Collect points as you survive
5. The player will have 3 hearts. Game ends when all hearts are lost.
6. Enjoy the game!

------------------------------
Notes
------------------------------
- The game uses `pgzrun` to handle the game loop and rendering.
- The `random` library is used to randomly generate obstacles and monster movement.
- Make sure all images used (player, player_f, player_left, player_right) are in the same folder as game.py.

==============================
Have fun playing!
==============================
